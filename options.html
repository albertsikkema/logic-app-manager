GitHub Configuration

How to create a GitHub Personal Access Token:
  1. Go to GitHub Settings → Personal access tokens → Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Set name: "Logic App Manager"
  4. Set expiration: 90 days (recommended)
  5. Select scope: repo (Full control of private repositories)
  6. Click "Generate token" and copy it immediately
Your token is stored securely and cleared when you close the browser
Format: owner/repository (e.g., mycompany/logic-apps-backup)
